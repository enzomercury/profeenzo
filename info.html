Programar es el proceso de escribir instrucciones para que una computadora realice una tarea. Estas instrucciones se dan a través de lenguajes de programación como Python, JavaScript o Scratch.

Es una habilidad poderosa que permite automatizar procesos, crear videojuegos, desarrollar aplicaciones móviles, diseñar sitios web y mucho más.

En el grupo 9°B, la programación no es solo una herramienta técnica, sino una forma de pensar, crear y resolver problemas del mundo real con inteligencia y empatía.
